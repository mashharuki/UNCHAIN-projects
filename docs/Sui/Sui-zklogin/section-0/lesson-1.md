# 0.1 プロジェクト概要 👋

この教材では、Suiの開発元であるMysten labs社が開発したzkLoginを使ったweb3アプリ開発をステップバイステップで学びます。

- **学習ゴール**：Google OAuthでWeb2ログイン→JWT取得→Suiアドレス生成→dApp操作までを体験
- **技術要素**：Sui、zkLogin、OAuth、JWT、Supabase、NFT
